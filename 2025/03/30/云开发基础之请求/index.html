更新云数据库的两种方法

update 局部更新一个或多个记录使用 update 方法可以局部更新一个记录或一个集合中的记录，局部更新意味着只有指定的字段会得到更新，其他字段不受影响。如果需要更新多个数据，需在 Server 端进行操作（云函数），在 where 语句后同样的调用 update 方法即可
set 替换更新一个记录 如果需要替换更新一条记录，可以在记录上使用 set 方法，替换更新意味着用传入的对象替换指定的记录,其他的就消失，只有重置的。如果指定 ID 的记录不存在，则会自动创建该记录，该记录将拥有指定的 ID。
