• 使用 display: flex 和 justify-content: space-around 确保 tab 在底部均匀分布。

• 图标和文字通过 flex-direction: column 垂直排列，并居中对齐。

• 如果需要将图片固定在页面顶部，推荐使用 position: absolute 或 fixed 。

• 如果需要将图片放在容器的顶部，可以使用 flexbox 或 grid 布局。• 如果容器是普通布局，可以通过调整 margin 或 padding 来实现。

• display: flex; ：使用 Flexbox 布局，使得子元素（ .tab-item ）可以水平排列。

• position: fixed; ：将导航栏固定在页面底部，不会随页面滚动而消失。• bottom: 0; left: 0; width: 100%; ：确保导航栏覆盖整个页面底部。

• flex: 1; ：每个 Tab 项占据相等的宽度，确保导航栏被均匀分割。• display: flex; ：使用 Flexbox 布局，使得图标和文字垂直居中排列。

• flex-direction: column; ：将子元素（图标和文字）按垂直方向排列。• align-items: center; justify-content: center; ：确保图标和文字在水平和垂直方向上都居中。
